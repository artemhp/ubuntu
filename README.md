# Ubuntu Configuration

This repository tracks the configuration and adjustments made to my Ubuntu setup.

## Terminal Customization

### Look and Feel

*   **Goal:** A modern, dark terminal with green text for better readability and a clean aesthetic.
*   **Theme:** Built-in "Gray on Black" scheme.
*   **Color Palette:** Built-in "Solarized" palette.
*   **Font:** A popular open-source monospace font. Suggestion: **Fira Code**.

### Color Palette

*   **Background (Off-Black / "Pastel Black"):**
    *   **Charcoal:** `#1E1E1E` (A very dark gray)
    *   **Slate:** `#2C3E50` (A dark, slightly blue-tinted black)
    *   **Nord Night:** `#2E3440` (A desaturated dark blue-gray)
*   **Text (Green):**
    *   **Classic Terminal Green:** `#00FF00` (Bright and iconic)
    *   **Modern Lime:** `#A6E22E` (Vibrant lime green)
    *   **Soft Mint:** `#50FA7B` (A softer, minty green)
*   **Bold Text:**
    *   **Brighter Green:** `#33FF33` (A more vibrant green for emphasis)
    *   **Light Yellow:** `#FFFF00` (Strong contrast, pairs well with dark themes)
    *   **Soft Orange:** `#FFA500` (Warm highlight, good contrast)
